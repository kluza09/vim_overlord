# vim_overlord

## Quick Start
1. Backup your current setup:

   ```bash
   mv ~/.vim ~/.vim_bck
   mv ~/.vimrc ~/.vimrc_bck
   ```

2. Set up .vim directory:

   ```bash
   git clone https://github.com/kluza09/vim_overlord ~/.vim
   ```

3. Configure Vundle setup by running:

   ```bash
   ./config.sh
   ```

4. Open `vim` and run following command:
   ```vim
   :PluginInstall
   ```

5. **Enjoy!**
