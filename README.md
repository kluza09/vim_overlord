# vim_overlord

## Quick Start
0. Backup your current setup:

   ```bash
   mv ~/.vim ~/.vim_bck
   mv ~/.vimrc ~/.vimrc_bck
   ```

1. Set up .vim directory:

   ```bash
   git clone https://github.com/kluza09/vim_overlord ~/.vim
   ```

2. Configure Vundle setup by running:

   ```bash
   ./config.sh
   ```

3. Install required Plugins
open `vim` and run following command:
   ```vim
   :PluginInstall
   ```
